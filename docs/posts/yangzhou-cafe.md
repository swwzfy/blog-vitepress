# 扬州咖啡馆工作指南

定居扬州，常年在外写代码。测评了多家咖啡馆的 WiFi、插座、噪音和咖啡质量。扬州节奏慢、湖光绿荫，很适合静下心来干活。

<!-- more -->

## 评价标准

对远程工作者来说，咖啡馆好不好看四个维度：

| 维度 | 权重 | 说明 |
|------|------|------|
| 网速 | ⭐⭐⭐⭐⭐ | 下载 ≥ 50Mbps 才够用，上传别低于 10Mbps |
| 插座 | ⭐⭐⭐⭐ | 没插座等于限时营业，至少桌边要有 |
| 噪音 | ⭐⭐⭐ | 通话多的需要安静，写代码的可以容忍一些 |
| 续航 | ⭐⭐⭐ | 能待多久不被赶，一杯咖啡的合理停留时间 |

## 独立咖啡馆

### 吉舍 Cafe&Bar（梗子街店）

早 C 晚 A 模式，扬州较早做日咖夜酒的店。咖啡师兼酒保，特调有调酒的影子，层次丰富。

- **地址**: 广陵区梗子街 66 号
- **营业**: 12:00-18:00 / 19:00-24:00
- **插座**: 有，数量一般
- **噪音**: 低，下午人少
- **推荐**: 想喝特调又想工作的人
- **注意**: 咖啡偏淡，建议点单时备注加浓

### RIFF COFFREE（鉴真店）

露营风装修，天幕、蛋卷桌、露营椅。除了咖啡还有私酿果醋，老板是户外爱好者。

- **地址**: 邗区万科彩虹汇 M 栋 140 号
- **营业**: 08:00-19:00
- **插座**: 有
- **噪音**: 低
- **推荐**: 喜欢露营氛围的人，早上开门早
- **注意**: 不定时公休，去之前确认是否营业

### 半间咖啡馆（宋城名都店）

以运河为灵感，店内有一艘木船装饰。老板喜欢用各地特产做时令特调，会热情讲解成分。

- **地址**: 广陵区新古一巷 77 号
- **营业**: 09:00-18:00
- **插座**: 有
- **噪音**: 低，店内有猫有狗
- **推荐**: 喜欢创意特调、需要安静环境
- **注意**: 店狗会出去散步，偶尔带客人回来

### 舟楫咖啡

自家烘焙，豆单不停换，用 SOE 做咖啡。空间虽小但分区明确：吧台、日系座位区、天台、户外座椅。

- **地址**: 广陵区盐阜东路 13-9 号
- **营业**: 08:00-20:00
- **插座**: 有
- **噪音**: 中等偏低
- **推荐**: 咖啡爱好者，喜欢尝试不同豆子
- **注意**: 座位不多，高峰期可能没位

### TODAY'S BEAN 今日赏味咖啡馆

自家烘焙，豆单丰富，总有适合你口味的。小而温馨，适合久坐。

- **地址**: 广陵区观巷 41 号
- **营业**: 08:00-18:00
- **插座**: 有
- **噪音**: 低
- **推荐**: 长时间工作的首选之一
- **注意**: 店面不大，建议早去占位

## 连锁品牌

### 星巴克（各门店）

扬州门店不算多，但基本设施齐全。

- **网速**: 30-60Mbps
- **插座**: 约一半门店有
- **噪音**: 高，尤其是商场店
- **推荐**: 选非商场店，环境好一些
- **注意**: 普通门店别指望太多，当应急用

### Manner

性价比高，美式 15 块，自带杯减 5 块。

- **网速**: 30-50Mbps
- **插座**: 新门店基本有
- **噪音**: 中等
- **推荐**: 预算有限的远程工作者
- **注意**: 座位少，高峰期占不到位

### 瑞幸咖啡

便宜、方便、覆盖广。但工作体验一般。

- **网速**: 20-40Mbps
- **插座**: 少
- **噪音**: 中等偏高
- **推荐**: 买杯咖啡坐一小时，不适合久坐
- **注意**: 座位设计偏休闲，不适合长时间办公

## 特色空间

### 灼灼鬖鬖咖啡厅（大运塔）

在中国大运河博物馆大运塔一层，艺术氛围拉满。配备意大利进口 LaMarzocco 咖啡机，专业水准。

- **地址**: 中国大运河博物馆大运塔一层
- **网速**: 不详，博物馆区域应该有公共 WiFi
- **插座**: 不详
- **噪音**: 低，博物馆环境安静
- **推荐**: 想在艺术空间工作的人
- **注意**: 需要先参观博物馆，适合半日游+工作

## 避坑指南

### 别去的类型

- **商场咖啡馆**: 噪音大、网速差、座位不舒服
- **网红打卡店**: 拍照的人比工作的人多，环境嘈杂
- **纯外卖店**: 没座位或者座位极不舒服

### 最佳时段

| 时段 | 推荐度 | 说明 |
|------|--------|------|
| 9:00-11:00 | ⭐⭐⭐⭐⭐ | 人少、安静、有座位 |
| 11:00-14:00 | ⭐⭐ | 午餐高峰，人多嘈杂 |
| 14:00-17:00 | ⭐⭐⭐ | 人渐渐多，但还行 |
| 17:00-19:00 | ⭐⭐⭐⭐ | 下班高峰前的空档 |
| 19:00 以后 | ⭐⭐⭐⭐⭐ | 人少，但有些店会关 |

### 必备装备

- **降噪耳机**: 咖啡馆工作第一生产力
- **充电宝**: 插座不够时的救命稻草
- **鼠标垫**: 有些桌子太滑，鼠标飘
- **手机热点**: WiFi 挂了的 Plan B
